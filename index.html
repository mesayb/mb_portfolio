About Me

My name is Mesay, I am originally from Ethiopia. I currently work as software automation tester for a Financial Service Company located in Alpharetta, Georgia. I attended my undergraduate study at Addis Ababa University and graduated in 2011 with a BSc in Chemical Engineering. After graduation I was more interested in Software Development Life Cycle and started working as a manual software tester.
As an automation tester my area of focus is on Graphic User Interface and API test automation. I have experience using tools like Cucumber, Selenium for UI automation and Mocha, Postman, Runscope, APIFortress for API testing. I am currently attending Full Stack Webdevelopment Boot Camp at Georgia Tech.
